La galería de The Ray Optics

Esto es una colección de simulaciones hechas usando the web app, que pueden ser usadas como demostraciones interactivas de óptica. Eres bienvenido de contribuir con nuevas simulaciones. Véase the contribution guide para más información. Para reportar un fallo, por favor open an issue on GitHub o envía un email a ray-optics@phydemo.app.

A Galeria de Óptica de Raios

Essa é uma coleção de simulações feitas utilizando o app web, que podem ser utilizadas como demonstrações interativas de óptica. Contribuições de novos itens são bem-vindas. Veja O guia de contribuição para mais informação. Para reportar um erro, por favor abra um issue no GitHub ou mande um email para ray-optics@phydemo.app.

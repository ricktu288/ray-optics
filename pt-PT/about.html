Simulação de Ótica de Raios

Uma aplicação web grátis e de código aberto para criação e simulação de cenas de ótica geométrica 2D.
v5.2+20250920.ecf2057

Este projeto, incluindo o conteúdo da galeria, é licenciado sob a Licença Apache 2.0 e é oficialmente hospedado no site PhyDemo(phydemo.app) (anteriormente em ricktu288.github.io). O código fonte está disponível no GitHub e contribuições são bem-vindas. Se houver questões ou sugestões, pode abrir issues ou discussões no GitHub, ou enviar um email para ray-optics@phydemo.app se não utilizar o GitHub.

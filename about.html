Ray Optics Simulation

A free, open-source web app for creating and simulating 2D geometric optical scenes.

Our goal is to make it easy for students to build physical intuition by "playing around", for teachers to do dynamical demonstrations, and at the same time also include tools for more advanced usage. The source code is available on GitHub, licensed under the Apache License, Version 2.0.

If you have questions or suggestions about this project, you can either open issues on GitHub or email yttu@duck.com (Yi-Ting Tu). You are also welcome to contribute to this project directly. If you use this project in an academic paper, please cite it using the metadata described here.

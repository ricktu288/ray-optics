Галерея лучевой оптики

Это коллекция симуляций, созданных с помощью веб-приложения, которые можно использовать в качестве интерактивных демонстраций оптики. Вы можете вносить новые элементы. См. руководство по внесению вклада для получения дополнительной информации. Чтобы сообщить об ошибке, откройте проблему на GitHub или отправьте электронное письмо по адресу ray-optics@phydemo.app.

ອຸປະກອນແສງເລຂາ

ນີ້ຄືການລວບລວມແບບຈຳລອງທີ່ສ້າງຂຶ້ນໂດຍໃຊ້ ແອັບເວັບ ເຊິ່ງສາມາດໃຊ້ເປັນການສາທິດແບບໂຕ້ຕອບກ່ຽວກັບແສງ. ຍິນດີຕ້ອນຮັບຖ້າທ່ານຕ້ອງການປະກອບສ່ວນເພີ່ມລາຍການໃໝ່. ເບິ່ງ ຄຳແນະນຳການປະກອບສ່ວນ ສຳລັບຂໍ້ມູນເພີ່ມເຕີມ. ຫາກຕ້ອງການແຈ້ງຂໍ້ຜິດພາດ, ກະລຸນາ ເປີດບັນຫາໃນ GitHub ຫຼື ສົ່ງອີເມວຫາ ray-optics@phydemo.app.
